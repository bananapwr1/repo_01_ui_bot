# Telegram UI Bot (Bot #1)

Простой пользовательский интерфейс для торговых сигналов.

## Функции:
- Команды `/start`, `/status`, `/plans`
- FSM для привязки PO-аккаунтов
- SQLite для хранения пользовательских данных
- Запись запросов на сигналы в Supabase

## Установка:
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Настроить `.env` файл
4. Запустить: `python main.py`

## Deploy на Bothost:
- Загрузить все файлы
- Установить переменные окружения
- Запустить бота
